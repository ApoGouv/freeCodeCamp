Introduction

Java is a high-level, class-based, object-oriented programming language designed for portability and efficiency. It is widely used for building web applications, enterprise software, mobile applications, and more.

Getting Started

To write and run Java programs, you need:

  • JDK (Java Development Kit): Includes the compiler (javac) and runtime (java).
  • IDE (Optional): IntelliJ IDEA, Eclipse, or VS Code with Java extensions.
  • Basic Syntax: Java programs are written in .java files and executed as .class bytecode.

Example:


public class HelloWorld {
  public static void main(String[] args) {
      System.out.println("Hello, World!");
  }
}
Variables and Data Types

Java is a statically-typed language, meaning variable types must be declared.


int myNumber = 10;
double myDecimal = 5.5;
char myChar = 'A';
boolean isJavaFun = true;
String message = "Java is great!";
If-Else Statements

int number = 10;
if (number > 5) {
    System.out.println("Number is greater than 5");
} else {
    System.out.println("Number is 5 or less");
}

In this example, an integer variable number is checked using an if-else condition. Depending on its value, a different message is printed to the console.

The condition checks if the number is greater than 5 and prints the appropriate message accordingly.

Loops

for (int i = 0; i < 5; i++) {
    System.out.println("Iteration: " + i);
}

int count = 0;
while (count < 5) {
    System.out.println("Count: " + count);
    count++;
}

The for loop is used to iterate a specific number of times, printing the iteration count each time.

The while loop, on the other hand, will keep running until the count variable reaches 5, incrementing the count with each iteration.

Methods

Methods allow code reuse and modularization in Java.


class Calculator {
    int add(int a, int b) {
        return a + b;
    }
    
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        System.out.println(calc.add(5, 3)); // Output: 8
    }
}
Classes and Objects

Java follows OOP principles, such as encapsulation, inheritance, and polymorphism.


class Car {
    String model;

    Car(String model) {
        this.model = model;
    }

    void drive() {
        System.out.println(model + " is driving.");
    }
}

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car("Tesla");
        myCar.drive();
    }
}
Inheritance

class Animal {
    void makeSound() {
        System.out.println("Animal makes a sound");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.makeSound();
        myDog.bark();
    }
}

Inheritance allows a class to inherit properties and methods from another class. In this example, the Dog class inherits the makeSound() method from the Animal class, while also adding its own bark() method.

  • Superclass: The class being inherited from (e.g., Animal).
  • Subclass: The class that inherits properties and methods (e.g., Dog).
  • Method Overriding: Subclasses can modify methods from the superclass for their own behavior.
Exception Handling

Java uses try-catch blocks to handle exceptions.


try {
    int result = 10 / 0; // Division by zero
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero!");
} finally {
    System.out.println("Execution finished.");
}
